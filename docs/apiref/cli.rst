Scaleout Edge CLI
=================

The Scaleout Edge Command-Line Interface (CLI) provides tools for managing
projects, clients, compute packages, and training sessions from the terminal.
It is designed to simplify common operations such as starting clients, uploading
models, monitoring training progress, and interacting with the Scaleout Edge
control plane.
